# 记忆系统

Claude Code 的记忆系统（memdir）正是为此而生：一个基于文件的、类型化的、跨会话持久的记忆架构。

1. 记忆类型：user、feedback、project、reference
    * 理念：只保存不可从当前项目状态推导的信息。
    * user：用户的角色、偏好、知识背景时。处理解释、协作的方式。
    * feedback：记录用户对 Agent 行为的纠正和确认。
    * project：项目状态。决策、截止日期、正在进行的工作。
    * reference：外部引用，外部哪些资源可以被利用。
2. 格式：markdown + YAML Frontmatter声明元数据。
    * 在这个 .md 文件的最顶部，用一对 --- 包裹起来的那块区域，里面写的全是 YAML 格式的键值对
    * MEMORY.md 是记忆系统的入口点——它不是记忆本身，而是一个索引文件。最多 200 行，最大 25KB
